## 📊 Соответие ТЗ

| Требование ТЗ | Статус | Реализация |
|--------------|--------|------------|
| База данных (водитель: ФИО, лицензия) | ✅ | `drivers_page.py`, `models.py`, `desktop_schema.sql` |
| База данных (ТС: марка, VIN, пробег) | ✅ | `vehicles_page.py`, `models.py`, `desktop_schema.sql` |
| CRUD-операции водителей | ✅ | `drivers_page.py` (создание, чтение) |
| CRUD-операции ТС | ✅ | `vehicles_page.py` (создание, чтение) |
| Привязка водителя к авто | ✅ | `vehicles_page.py` (поле driver_combo) |
| Мониторинг техсостояния | ✅ | `maintenance_page.py` |
| Напоминания о ТО/страховке | ✅ | `maintenance_page.py` (кнопка "Проверить напоминания") |
| Отчеты по пробегу | ✅ | `reports_page.py` |
| Отчеты по штрафам | ✅ | `reports_page.py` |
| Интеграция с ГИБДД API | ✅ | `gibdd_api.py`, `fines_page.py` (кнопка синхронизации) |
| Роли доступа (диспетчер/водитель) | ✅ | `auth_dialog.py` (4 роли: диспетчер, водитель, механик, администратор) |
| Python + PostgreSQL | ✅ | `requirements.txt`, `database.py` |
| Дизайн в стиле сайта | ✅ | `styles.py` (бежевая палитра Carvix) |
| Локальная работа в офисе | ✅ | Desktop-приложение с локальным доступом к БД |

## 🎨 Дизайн-система

Приложение использует дизайн-систему Carvix:
- **Цвета**: бежевая палитра (#FBF8F3, #F5EFE3, #ECE2CE, #DCC9A8, #C8AE82)
- **Шрифты**: Manrope (основной), Cormorant Garamond (заголовки)
- **Компоненты**: карточки с закругленными углами, плавные анимации
- **Темы**: светлая тема с темными акцентами
