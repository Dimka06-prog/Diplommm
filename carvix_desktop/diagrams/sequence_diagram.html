Диаграмма последовательности авторизации

User
Пользователь
AuthDialog
Диалог входа
Database
База данных
Logger
Логгер
1. Ввод логина/пароля
User вводит учетные данные в AuthDialog
2. authenticate()
AuthDialog вызывает метод аутентификации
3. SELECT * FROM sotrudnik WHERE login = ?
AuthDialog запрашивает данные пользователя из Database
4. Возвращает данные пользователя
Database возвращает данные пользователя
5. Проверка пароля (bcrypt)
AuthDialog проверяет пароль
6. log_login()
AuthDialog логирует успешный вход в Logger
7. authenticated.emit()
AuthDialog отправляет сигнал аутентификации
8. accept()
AuthDialog закрывает диалог с успешным результатом
